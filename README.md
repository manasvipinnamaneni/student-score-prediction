# Student Score Prediction using Machine Learning

A beginner-friendly Machine Learning project that predicts student exam scores based on:
- Study Hours
- Attendance Percentage
- Sleep Hours

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Project Structure

student-score-prediction/
│
├── main.py
├── data.csv
├── requirements.txt
└── README.md

## Dataset
The dataset contains:
- Hours Studied
- Attendance
- Sleep Hours
- Exam Score

## Machine Learning Algorithm
- Linear Regression

## Features
- Loads dataset from CSV file
- Trains ML model
- Accepts user input
- Predicts exam score
